Problem No-1 Move Characters to Front:

Task i: Given a string of length n containing some # characters, move all hashes to the start of the string.
Example: Input Move#Hash#to#Front 
 Output ###MoveHashtoFront.

Task ii: move uppercase characters to the start of the string
Task iii: move non-alphabetic characters to the start of the string
Task iv: move vowels to the start of the string 

No-2 String Compression:
Task: Compress consecutive repeating characters by following them with their count.
Example: Input aabbbbeeeeffggg 
 Output a2b4e4f2g3.

No-3 Custom Exception Handling:
Task: Create a class to handle specific business logic errors, such as a NegativeValueException when calculating the sum of digits of a number. 

No-4 
Task: Create a MaterialException that throws specific messages if input values (like mass) are negative or zero.

No - 5 Employee Filter:
Problem: Given a list of Employee objects, find the names of all employees whose salary is greater than 50,000 and who belong to the "IT" department.

No- 6 Find First Non-Repeating Character:
Problem: Use a Stream to find the first character in a string that does not repeat.
Logic: Convert the string to a stream of chars, use groupingBy with LinkedHashMap to maintain order, and find the first entry with a count of 1.

No- 7 Summary Statistics:
Problem: Find the Maximum, Minimum, and Average of a list of integers using a single Stream operation.
Solution: Use IntSummaryStatistics via stream.mapToInt(x -> x).summaryStatistics().

No - 8 Remove Duplicate Words:
Task: Given a sentence, remove all duplicate words and print the unique ones in their original order.
Focus: Use LinkedHashSet or Java 8 Streams with distinct().

No- 9 Vowel Count and Sort:
Task: Take a list of strings and sort them based on the total number of vowels they contain (descending order).
Focus: Custom Comparator with stream().sorted().

No- 10 Employee Grouping:
Task: Given a list of Employee objects (ID, Name, Salary, City), group them by their City.
Focus: Use Collectors.groupingBy().  

No- 11 Find Second Highest Salary:
Task: Use the Stream API to find the second highest salary from a list of Integers or Employee objects.
Logic: .stream().distinct().sorted(Comparator.reverseOrder()).skip(1).findFirst().

No- 12 The Ticket Counter (Queue Logic):
Task: A queue of people has specific "priorities." If a person with higher  priority arrives, they move to the front.
Focus: Implementing a PriorityQueue or managing a LinkedList.

No- 13 Product Stock Update:
Task: You have two lists: CurrentInventory and NewArrivals. Update the quantity of    products and add new ones to the list.
Focus: Map operations (merge or putIfAbsent). 

No - 14 Equilibrium Index:
Task: Find the index in an array where the sum of elements to the left is equal to the sum of elements to the right.
Example: Input [1, 2, 3, 4, 6] → Output Index 3 (Sum \(1+2+3=6\)).

No - 15 Move Zeros to End:
Task: Move all zeros in an array to the end while maintaining the relative order of non-zero elements.
Example: [0, 1, 0, 3, 12] → [1, 3, 12, 0, 0].
